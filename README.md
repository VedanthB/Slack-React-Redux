# Slack-React-Redux

A clone of the slack app. Made with react / redux / firebase. you can create rooms and send messages in that room. Sign in with google

Link to live app: https://slack-clone-99c4d.web.app


# Tech used =>

- React
- Readux 
- Firebase
- styled-components
- Frebase-hooks 
- react-router

- Hosted on firebase
- lazy loading
